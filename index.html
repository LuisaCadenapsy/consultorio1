La integración directa con Google Calendar requiere que esta app esté publicada en tu propio dominio (GitHub Pages, por ejemplo). Sigue los pasos abajo para configurarla.
Paso a paso: conectar Google Calendar
1
Publica esta app en GitHub Pages
Crea una cuenta en github.com → New repository → nombre: consultorio → sube este archivo HTML como index.html → Settings → Pages → Branch: main → Save. Tu URL será tunombre.github.io/consultorio.
2
Crea credenciales en Google Cloud
Ve a console.cloud.google.com → Nuevo proyecto → APIs y servicios → Habilitar APIs → busca "Google Calendar API" → habilitar. Luego: Credenciales → Crear credencial → ID de cliente OAuth → Aplicación web. En "Orígenes autorizados" agrega tu URL de GitHub Pages. Copia el Client ID.
3
Pega tu Client ID aquí
Una vez publicada la app en GitHub Pages, vuelve a esta pantalla y conecta.
4
Importar citas del calendario
Una vez conectada, tus próximas citas aparecerán aquí para importarlas como sesiones con un clic.
Importar citas manualmente

Pega el texto de tus citas (copiado de Google Calendar o cualquier fuente) y conviértelas en sesiones.
